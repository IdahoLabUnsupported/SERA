SERA:
Simulation Environment for Radiotherapy Applications
User's Manual
Version 1C0

D. E. Wessol,  C.A. Wemple, F.J. Wheeler, M.T. Cohen, M.B. Rossmeier, and Joshua Cogliati
December 11, 2001

Summary

SERA is a suite of command line or interactively cross-launched software modules for developing Boron Neutron Capture Therapy (BNCT) patient treatment plans.  One of SERA's main functions is to provide a graphical environment for constructing  and displaying uniform volume-element-based solid models derived from medical images.  These solid models form the geometric basis of the BNCT target areas for the BNCT radiation transport model.  Additionally, SERA provides the environment for executing the radiation transport calculation and displaying the results of that calculation.

INEEL is initially only providing full support on a limited number of systems with 24-bit graphics capabilities in order to best support viewing, manipulating, and sampling three-dimensional models of imaged anatomical features.  Check the URL http://www.cs.montana.edu/~bnct for a current list of actively supported systems. Images are accepted in a  number of different formats, including raw 8-bit, dicom3,  Qsh, and other formats. The Qsh format is used as a the Sera internal image format.  Tools are supplied to convert the various image formats to Qsh.  Various image processing and surface and volume rendering tools are also supplied.

SERA can be interactively driven, and the interface is based on the portable X11 window environment with the Motif tool kit and the OpenGL graphics libraries.

License Requirements

The use of SERA will require a contractual agreement with the Idaho National Engineering and Environmental Laboratory for the composite software system.
